# News API

Klasa `NewsAPI` jest gotowa i dość dobrze opisana. Główna metoda to `watch`, która na bieżąco pobiera info z api na temat podanego słowa kluczowego.

Obsługiwane serwisy (gotowe scrapery): bbc-news, cnn
